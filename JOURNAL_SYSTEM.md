# Journal System

==================================================
Journal Overview
==================================================

Journal ใช้สำหรับ:
- บันทึกเนื้อเรื่อง
- จด Hint
- บันทึกข้อมูลสำคัญ
- รองรับ CRUD

==================================================
Journal Types
==================================================

1. Story Journal
2. Player Journal

==================================================
Story Journal
==================================================

ระบบสร้างให้อัตโนมัติ

ใช้สำหรับ:
- Lore
- Story
- Boss Hint
- Event Hint

ผู้เล่นสามารถ:
- Read ได้อย่างเดียว

==================================================
Player Journal
==================================================

ผู้เล่นสร้างเองได้

ใช้สำหรับ:
- จด Hint
- จดทางเลือก
- จดข้อมูลสำคัญ

รองรับ:
- Create
- Read
- Update
- Delete

==================================================
Journal Database Fields
==================================================

- JournalId
- FloorNumber
- JournalType
- Title
- Content
- CreatedAt
